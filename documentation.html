Panduan Instalasi Wazuh Agent dan YARA

Versi 1.0 | November 2025

Pendahuluan

Panduan ini menyediakan langkah-langkah lengkap untuk menginstal Wazuh Agent versi 4.13.0 dan YARA di sistem Anda. Wazuh Agent berfungsi sebagai agen monitoring yang mengumpulkan informasi keamanan, sementara YARA adalah mesin pendeteksi malware berbasis aturan.

Bagian 1: Instalasi YARA

1.1 Persyaratan Sistem

OS yang Didukung:
• Ubuntu/Debian
• CentOS/RHEL/Fedora
• AlmaLinux/Rocky Linux

1.2 Repository YARA

YARA akan diinstal dari repository GitHub:

https://github.com/kaivanriz/yaraxdr.git

1.3 Langkah-langkah Instalasi

Langkah 1: Clone repository YARA

git clone https://github.com/kaivanriz/yaraxdr.git

Langkah 2: Masuk ke direktori YARA

cd yaraxdr

Langkah 3: Baca dokumentasi lengkap

Dokumentasi detail tersedia di repository. Ikuti instruksi instalasi sesuai README.md untuk konfigurasi spesifik sistem Anda.

Dokumentasi:
https://github.com/kaivanriz/yaraxdr/blob/main/README.md

1.4 Verifikasi Instalasi YARA

Setelah instalasi selesai, verifikasi dengan perintah:

yara --version

Bagian 2: Instalasi Wazuh Agent 4.13.0

2.1 Informasi Konfigurasi

Parameter Nilai
Versi Wazuh Agent 4.13.0
Manager IP Address 172.16.10.196
Tipe Agent x86_64 (Linux)

2.2 Instalasi untuk CentOS, RHEL, Fedora, AlmaLinux, Rocky Linux

Langkah 1: Download package Wazuh Agent

curl -o wazuh-agent-4.13.0-1.x86_64.rpm https://packages.wazuh.com/4.x/yum/wazuh-agent-4.13.0-1.x86_64.rpm

Langkah 2: Instal dengan konfigurasi manager

sudo WAZUH_MANAGER='172.16.10.196' rpm -ihv wazuh-agent-4.13.0-1.x86_64.rpm

2.3 Instalasi untuk Ubuntu dan Debian

Langkah 1: Download package Wazuh Agent

wget https://packages.wazuh.com/4.x/apt/pool/main/w/wazuh-agent/wazuh-agent_4.13.0-1_amd64.deb

Langkah 2: Instal dengan konfigurasi manager

sudo WAZUH_MANAGER='172.16.10.196' dpkg -i ./wazuh-agent_4.13.0-1_amd64.deb

2.4 Verifikasi Instalasi Wazuh Agent

Setelah instalasi selesai, verifikasi status agent:

sudo systemctl status wazuh-agent
Status Berhasil: Agent harus menunjukkan status active (running)

Bagian 3: Konfigurasi Lanjutan

3.1 Lokasi File Konfigurasi

Path konfigurasi Wazuh:
/var/ossec/etc/ossec.conf

3.2 Menjalankan Wazuh Agent

sudo systemctl start wazuh-agent sudo systemctl enable wazuh-agent

3.3 Melihat Log Agent

sudo tail -f /var/ossec/logs/ossec.log

Bagian 4: Troubleshooting

4.1 Masalah Koneksi Manager

Jika agent tidak terhubung ke manager:

1. Verifikasi IP manager: 172.16.10.196
2. Pastikan firewall tidak memblokir port komunikasi (default: 1514)
3. Periksa status layanan: sudo systemctl status wazuh-agent
4. Lihat log error: sudo tail -f /var/ossec/logs/ossec.log

4.2 Restart Agent

sudo systemctl restart wazuh-agent

4.3 Uninstal Agent

Untuk RHEL/CentOS:

sudo rpm -e wazuh-agent

Untuk Ubuntu/Debian:

sudo dpkg -r wazuh-agent

Bagian 6: Checklist Instalasi

  • ☐ YARA berhasil diunduh dan diinstal
  • ☐ Verifikasi YARA dengan perintah yara --version
  • ☐ Wazuh Agent 4.13.0 berhasil diunduh
  • ☐ Manager IP dikonfigurasi ke 172.16.10.196
  • ☐ Agent berhasil diinstal
  • ☐ Status agent menunjukkan active (running)
  • ☐ Konfigurasi firewall memungkinkan komunikasi
  • ☐ Agent terhubung ke manager
